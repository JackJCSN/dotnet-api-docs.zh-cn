> [!NOTE]
> **.NET Core 仅在 Linux 和 macOS 系统上运行：** C 和 Posix 区域性的排序规则行为始终区分大小写，因为这些区域性不使用预期的 Unicode 排序规则顺序。 建议使用除 C 或 Posix 以外的区域性执行区分区域性但不区分大小写的排序操作。
